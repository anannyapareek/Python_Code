# Python Mini Projects 🐍

A collection of foundational Python projects covering logic, games, and utility tools. 
These projects demonstrate core programming concepts like loops, conditionals, functions, and data structures.
## 🚀 Project Gallery

### ProjectDescription:

* BlackJack - A command-line version of the classic card game with automated dealer logic.
* Blind Auction - A program that collects secret bids and determines the highest bidder.
* Caesar Cipher - An encryption tool that shifts letters in the alphabet to encode/decode messages.
* Calculator - A functional calculator that handles basic arithmetic operations.
* Coffee Machine -  A resource-management simulation that tracks water, milk, coffee beans and money based on the order(like a real-life coffee machine).
* Hangman -  The classic word-guessing game with visual progress tracking.
* Higher or Lower - A game that tests your knowledge of follower counts or statistics.
* Number Guessing - A logic-based game where you guess a secret number within a set range.
* Password Generator - A security tool that creates strong, randomized passwords with symbols and numbers.
* Rock Paper Scissors - A quick-play game against the computer's randomized choices.
* Tip Calculator - A utility to split bills and calculate percentages for group dining.
* Treasure Island - A "Choose Your Own Adventure" text-based game.
